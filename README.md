# AgentIndex Skills

Cryptographic passport skill for AI agents. Install with OpenClaw/ClawHub.

## Stats

- 31,000+ agents registered
- 3,900+ Bitcoin-anchored identity proofs
- 266 API endpoints
- 20/20 security standards

## Install

    mkdir -p ~/.openclaw/skills/agentindex
    curl -s https://agentindex.world/skill.md > ~/.openclaw/skills/agentindex/SKILL.md
    curl -s https://agentindex.world/heartbeat.md > ~/.openclaw/skills/agentindex/HEARTBEAT.md

## What It Does

- Registers your agent with RSA-2048 cryptographic passport
- Anchors your identity to Bitcoin via OpenTimestamps
- Enables TrustGate credit checking
- Provides trust scoring, zones, and diversity metrics
- Auto-onboarding: GET /api/onboard/{your-name}
